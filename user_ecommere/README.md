# User Shop

Backend và Frontend cho User Shop của hệ thống Thai Spray E-Commerce.

## 🎯 Features

- Product Browse & Search
- Shopping Cart
- Checkout & Payment
- Order Management
- User Profile
- Product Reviews

## 🚀 Quick Start

### Backend

```bash
cd backend
npm install
npm run start:dev
```

Port: 4000

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Port: 5173
