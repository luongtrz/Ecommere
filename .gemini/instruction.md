# Project Instructions & Context

## Design Preferences
- **Admin Forms**: Prefer full-width (`w-full max-w-[98%]`) layouts to maximize screen real estate.
- **Form Layout**: Use 2-column grid layouts (50/50 split or similar) for main content to reduce vertical scrolling.
- **Spacing**: Use compact spacing (`gap-4`, `space-y-4`, small paddings) in admin interfaces to keep content dense and viewable without excessive scrolling.
- **Images**: Ensure image upload sections are prominent and large enough.
